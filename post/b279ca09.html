本网站正式投入使用

发表于2021-02-23|更新于2021-04-05
|字数总计:983|阅读时长:2分钟|阅读量:

这是我这三天以来，呕心沥血的成果了

这个网站的搭建，太费脑子了，主要是我压根不会服务器这些，也只能看得懂一点点的html代码，然而，做一个博客要用到的前端知识远不止这些。

准备工作

我一开始在知乎找的教程，后来才知道大错特错，可能碍于知乎的编辑，就没有那么详细，步骤都是跳着来的。导致我每次error的时候，都要再花上一两小时的时间去排查错误，这是很残酷的一点，CSDN上的教程都还行，起码评论区有人问题跟我一样，我才知道我是什么问题

  • Node.js
  • Git
  • Hexo

特别是Git，我一开始的指令全是打在cmd里面的，要不是看到是在文件夹里面Git，我可能还以为是我的电脑的问题，每次都是

系统找不到指定文件

后来我终于装好了Git之后，最难的其实是Hexo的安装吧，我在Git的cmd里面，像各大教程输入的指令，一半以上显示的都是 error 和 warning 。

搭建完大概了之后

服务器正常上架，没错，就是那个最丑的Hexo模样，为了有一个好主题，我又是一顿找教程。事实证明，如果能看着视频学东西，绝对比看着字面的要强得多。
感谢b站
虽然我这次压根没在b站上搜教程（昨晚去b站搜了一下，果然Hexo换主题的教程没让我失望，可惜我已经踩着坑傻傻地闯进来了）

更换主题+二次装修

啊没错，又是在CSDN博客上面找的教程，然后…硬生生改了几小时没改好。本来加了几个.css到网页里面，结果直接以文本形式贴在了网页上，极其可悲，虽然后来终于解决了这个问题（本地端上传到Github上有点延迟导致的）
然后去b站找了一下主题配置方面的教程后我觉得我又行了，将会有更多更丰富的装饰更新，在几个工作日内

工作日：我可以自由地操作我的电脑的日子，节假日除外

最后的域名

昨天晚上，2021年2月22日，我在阿里云的域名购买并审核通过（终于），然后就解析上路了

www.yiang.store

虽然解析个域名也耗费了很多精力，但是一直刷新过后，页面加载出来的适合时候，噢！！！！！

发布文章

这篇是我的第一篇文章in my blog。那篇hello word不是，那是系统自己生成的，我怎么可能会写hello word呢我又不是程序员
突然想起一个笑话

一个程序员拿起了文房四宝
第一个写下的字就是
HelloWord

然后写的文章还是得编辑markdown文件（.md）。
梦回 visual studio
md文件的编辑目前也只是刚接触，不过很容易，反正整一编辑器，啥不是一样写，写东西啊，难的还在人心

终于，我的博客搭建好了，其实现在拿还有人用这种方式表达自己呢，QQ动态也好，微信朋友圈也好，知乎也好，谁还不缺一个地方展示自己。
我也不缺。
现在我们哪会缺工具缺平台，缺的一直是没有写东西的习惯，或者是思考的能力
不过我也不是想要一个表现的机会吧，我只想要有个清净的地方

至于以后我会写什么，还得看以后造化
不过，反正这是我自己的博客了，写什么还不是随心所欲，会看的人自然会看，不看的人不会打扰
这就是Yiang
