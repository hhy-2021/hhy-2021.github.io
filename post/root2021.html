压榨我的旧手机//旧手机改变计划

发表于2021-08-06|更新于2023-01-27|工
|字数总计:1.5k|阅读时长:4分钟|阅读量:

从还没放暑假开始计划，今天终于开工了！

刷机前的准备

  • 在鸿蒙系统出了那段时间，看到关于HarmonyOs的一些信息
    【鸿蒙的源码集成和Linux和liteos两个内核】【安卓也是基于Linux开发】
  • 我在一开始打算建立网站的时候看过一些搭建服务器教程，一部分是Linux内核服务器（+宝塔控制面板）
  • 我的博客网站都是用命令行来控制，巧了 Linux也是
  • 比较久之前看一个白嫖 Minecraft 服务器的教程，在远程 Linux 的服务器里安装 Java来运行
  • 刚好我有一部旧手机
  • 我在 Github 上的文件传输都是 ssh 协议
  • 很早以前我就接触过 root

这些信息，刚好闭环了
我隐约感觉我的旧手机很有利用价值

所以我在CSDN博客以 “旧手机” 为关键词搜素了遍，把有用的东西加入到收藏夹吃灰后就没有打开了，直到今天才开始刷机

小细节：百度知道只会教你把手机设置成用亮当时钟摆着；知乎会教你怎么做个人云；而CSDN会教你怎么跑Linux服务器

开始刷机 | Root

手机为红米note7pro，很容易

理想状态

小米系手机，先解bl锁，然后进入fastboot，刷入第三方recovery，usb导入magisk包，进行安装，完成。很容易

实际上

解锁很成功

翻车

解锁后手机重启，连续重启了十来次然后进入了一个不知道什么模式，只有“清除数据”“重新启动”“进入小米社区”三个选项。只能用音量键+-控制上下，电源键确认。

懵逼了半分钟后我决定强刷recovery，刷入时还是很顺利的

然后问题就来了，无法从电脑导入magisk包。MTP协议连接时一直断。问题出在手机里面文件太多，解锁后数据作废但是并没有被清除导致电脑无法传输文件。

按道理只要清理手机文件就行了，于是在rec模式下格式化data。
结果，fail
到这里其实就很懵逼，格式化失败，电脑也传输不了文件
关键是手机还在rec

到这里，有点崩溃，因为前不着村后不着店。如果就这样废了我的手机，那我的科研梦将延后两三年等我的K40退役才能继续了

所以我决定自己乱搞一下。

救砖

我好歹是搞机玩家，只要手机还能亮，总得刷回来。

直接打开我的小米专业刷机软件（要不是之前在吾爱论坛看到有人发，估计今天我就死在这里了），下载了个稳定版MIUI直接刷回去。刷机过程等了半小时，电脑刷机软件进度条跑了半天不动，最后显示error

但是手机开起来了，MIUI 12.0.2 版本，开机开半天（感觉12.5.0版本还是比12.0.2开机要快的）

折腾半天，手机还退化了。不过幸运的是手机没挂

继续解锁，这次留了个心眼，开发者模式中开启usb选项全开了，然后继续解bl锁，解锁完手机是可以正常使用的，所以第一次应该是这里出来问题

然后就是激动人心的时刻，刷入面具magisk，获取超级用户权限root

安装 Linux

目前未安装成功，虽然root了，但很多软件在配置的时候还是各种error和fail
另外ssh配置估计也要好久

探索不止 | 回馈无知

想起了很久以前吧，第一次玩智能手机的时候玩植物大战僵尸2

有个boss老是打不过，然后找各种方法后决定开挂，用的葫芦娃修改器改钻石改金币。可惜只是改了个表象，实际还是没有变化的。后来改游戏速度，增加了反应时间和思考时间后成功过关。

后来在我的一部三星手机，因为手机太卡而且无法关闭自启动，我走上了root的路，最后那部手机就废了，一开机就显示一个对话框 “ Root大师已关闭 ” 然后自动关机

我那个时候没啥思考能力也没啥逻辑吧，对这些总是学不会，也没有电脑也没有别的手机给我玩。我也只不过想让我的手机能被我掌控权限

我的设备还拿不到最高权限那不是扯淡吗？在我的领土，都得由我掌握！

刚好昨天看b站，这个视频

以前的Android玩家：superSU，xposed，超频Kemel，adb，fastboot、9008、sp tools、CyanigenMod刷机救砖 样样精通（涉猎）、Alpha尝鲜（bug）版？冲！

现在的Android玩家：喵喵喵这个版本的miui又热又卡 隔壁又上了个禅定模式好好玩 Msgisk好麻烦啊啊啊啊 Google服务好耗电啊 呜呜呜手机网络延迟好大老是断流 什么垃圾软件？居然要钱？​

Root，我自己的主力手机都不敢的操作，更何况刷机。大多数时候的探索，更像是在追梦，追小时候怎么也无法满足的梦、以及探索本身带给我的快乐。

就像视频里面说的，刷机可能对生活起的作用微乎其微。但当把一部手机刷机成功，开机动画出现的那一刻，好像什么都值得了。不只是手机的探索，在我的博客网站完工的那一刻我也无比激动。而今天的故事，更像是在讲给很久很久以前那个小破孩听，他做不到的事、圆不了的梦，有我在探索着。
