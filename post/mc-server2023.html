搭建Minecraft游戏服务器

发表于2023-01-12|更新于2023-01-19|工
|字数总计:1k|阅读时长:3分钟|阅读量:

前言

那天凌晨把一切做好，已经两三点
突然觉得很神奇，初二那年想在电脑玩我的世界，不知道什么叫Java；大四的时候就开了属于自己的游戏服务器
测试的时候看着满屏黑色指令代码框，居然不陌生了啊。

搭建服务器过程

  • 服务器管理后台程序mcs
  • 服务器forge安装（不加mod没意思）
  • 本地运行服务端测试
  • 打包到云端
  • 测试云端

这次遇到的问题是，我忘了我装过Java，然后看的教程示例是ubantu，用apt安装Java。然后我就想我cent OS要yum安装，然后就发现
yum -y java //我记得是这样的指令
然后：安装失败

于是我就一直想怎么快速解决这个问题
绕了一圈等我知道怎么安装Java的时候发现原来服务器已经有了Java环境，而我最开始的
java -version 打多了一条-
所以。。。。别的一下子就完成了。mcs的开发者集成了一键安装命令，我就喜欢这样的。

别的就更容易了，基本没有再遇到难题~

唯一一个不知道是不是bug的，服务端的Java，给它分配4G最大内存，当我让它运行的时候，服务器直接崩了。宝塔后台都进不去，卡了大概5分钟，最后用阿里云的后台重启，修改了最大内存就解决了。

测压ing……

最后换了台机测试客户端，发现死活连不上服务器。查了一下发现是正版验证的问题，然而此时我已经关了它。
就在我百思不得其解的时候想到了重启大法，重启真的很重要哈哈哈，特别是修改了配置文件之后。

客户端和门户网站

对，要做就做全套！
这次，我筛选出了一个中低配机子效果最好的光影，嘎嘎香，也不知道是不是磷的作用。
当我把游戏核心文件夹和pcl2以及操作说明.txt打包上传到我的云盘的时候，只有一个感觉就是nice
其实想都不用想，当年我要玩mod，要加光影的时候，作为一个小白是不可能轻易上手的。所以作为服务器开发者，让一个用户能够使用自己的产品也是一种艺术。
当然，我还要让“用户”更快下载。
200M左右的包，百度云下载起码得十分钟吧？蓝奏云也用不了了吧？这个时候用自己的云盘外链，不仅高速下载而且在官网上进入的时候会发现域名是【pan.yiang.store】
遥遥领先！

然后是官网，

https://mcs.yiang.store

我花了10分钟把网站上线了，因为早在一年前我就已经找好了网页文件，稍微设计一下便是我的。不过昨晚的时候觉得它的icon不够好看换了一套，然后又加了一些东西。

一切到这里就差不多结束了部署阶段。其实没人会关心怎么部署服务器，我想大家更关心的应该是：

服务器世界–云之邦

在之前我和我哥的游戏存档也十分强大，但是，可以更好！服务器来玩Java版，与手机玩网易版对比，那这升级不是一点半点。
而游戏服务器可以更加拉进玩家们的现实距离，开放世界的联机，有点元宇宙那味儿。

等玩家稍微集结，将开一个庆典大会
等世界运行至周年庆、服务器日等等大型活动，也会召开全服大庆典

5天后……

服务器一天崩五六次，影响了我的京东和云盘等业务
换了1.16.5版本，然后存档换不过来就没了
云盘被高速下载时游戏服务器就会很卡所以也取消掉了

感觉还行，虽然没有预期那么豪华
